# SolFoundry Social Media Templates

5 social media design templates for community promotion.

## Templates

### 1. Bounty Announcement
- Size: 1200x630px (Twitter/LinkedIn)
- Features: Bounty title, reward amount, tier badge, CTA button
- Colors: Forge dark background, emerald accent

### 2. Developer Spotlight
- Size: 1080x1080px (Instagram)
- Features: Developer avatar, stats overlay, achievements
- Colors: Purple gradient background

### 3. Weekly Stats Roundup
- Size: 1200x675px (Twitter)
- Features: Total bounties, total paid, top hunters, chart
- Colors: Forge dark, data visualization in emerald/magenta

### 4. Milestone Celebration
- Size: 1080x1080px (Instagram/Twitter)
- Features: Large number/stat, celebration elements
- Colors: Magenta accent with confetti

### 5. Tutorial/Content Promo
- Size: 1200x630px (Twitter/LinkedIn)
- Features: Content preview, title, author, CTA
- Colors: Emerald primary, forge dark bg

## Files

- \`bounty-announcement.png\` - Bounty announcement template
- \`developer-spotlight.png\` - Developer spotlight template
- \`weekly-stats.png\` - Weekly stats roundup template
- \`milestone.png\` - Milestone celebration template
- \`tutorial-promo.png\` - Tutorial/content promo template

## How to Use

1. Copy the template you need
2. Replace placeholder text/images
3. Export at the specified size
4. Post to social media with relevant hashtags: #SolFoundry #BountyHunting #Solana

## Fonts

- **Headings**: Orbitron (brand font)
- **Body**: Inter (system font)
- **Monospace**: JetBrains Mono (stats/code)
