# SolFoundry Social Media Templates

Submission for issue #829: Social Media Templates.

## Deliverables

- 5 reusable bounty announcement template concepts
- Each template exported in:
  - 1080 x 1080 feed PNG
  - 1200 x 675 Twitter/X card PNG
- Editable source assets included as SVG files
- Text fields are intentionally structured around bounty title, reward amount, and tier

## Files

- `png/feed/`: square feed exports
- `png/twitter-card/`: Twitter/X card exports
- `svg/feed/`: editable square SVG source files
- `svg/twitter-card/`: editable card SVG source files
- `source/template_specs.json`: editable template copy and palette data
- `source/render_templates.py`: reproducible renderer for the PNG exports
- `preview-contact-sheet.png`: quick visual overview

## Template Set

1. Bounty Open
2. High Reward
3. Audit Call
4. Builder Spotlight
5. Launch Week

## Brand Notes

The set uses a dark forge-themed base with SolFoundry emerald, magenta, violet, cyan, and gold accents. Each layout keeps a clear editable area for future bounty titles, reward amounts, and tier labels.
