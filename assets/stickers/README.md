## Design Guidelines

- **Color Palette**: Use the SolFoundry design system colors
  - Emerald: #00E676 (primary)
  - Purple: #7C3AED (secondary)
  - Magenta: #E040FB (accent)
  - Forge Dark: #0A0A0F (background)
- **Style**: Flat design with subtle gradients, consistent with SolFoundry branding
- **Format**: SVG source, exported to PNG/WebP at 128x128, 256x256, 512x512

## Usage

These stickers can be used in:
- Discord server
- Telegram group
- GitHub comments
- Social media
