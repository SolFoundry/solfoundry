# SolFoundry Sticker Pack

Submission for issue #835: SolFoundry Sticker Pack.

## Deliverables

- 10 unique Telegram/Discord sticker designs
- Transparent PNG output
- 512 x 512 px minimum size
- Shared SolFoundry style: anvil, forge, developer culture, emerald and magenta accents

## Files

- `png/`: final transparent PNG sticker files
- `preview-contact-sheet.png`: overview image for quick review
- `source/sticker_specs.json`: editable sticker names, captions, colors, and motif mapping
- `source/render_stickers.py`: reproducible renderer used to generate the PNG files
