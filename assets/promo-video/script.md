# SolFoundry 30-Second AI Promo Video Script

## Video Overview
- **Duration**: 30 seconds
- **Style**: Motion graphics / animated
- **Resolution**: 1920x1080, 60fps
- **Music**: Upbeat electronic, royalty-free

## Script

### [0-3s] Opening - The Problem
**Visual**: Developer frustrated, looking at a boring issue tracker
**Text overlay**: "Tired of finding bounties manually?"
**Animation**: Screen glitch effect, transition

### [3-8s] Introduction
**Visual**: SolFoundry logo animates in with forge effect (sparks, embers)
**Voiceover**: "Welcome to SolFoundry"
**Text overlay**: "The AI-Powered Bounty Forge on Solana"
**Animation**: Logo assembles from code fragments

### [8-15s] How It Works
**Visual**: Three-step animated flow
1. **Post** → Developer creates bounty (card flies in from left)
2. **Code** → Contributor submits PR (code editor animates)
3. **Earn** → AI reviews and pays out (coins cascade)
**Text overlay**: "Post. Code. Earn."
**Animation**: Smooth transitions between steps

### [15-22s] Key Features
**Visual**: Rapid-fire feature showcase
- AI-powered review (robot eye scanning code)
- Multi-LLM analysis (three AI avatars)
- Instant payouts (Solana transaction animation)
- Reputation building (profile card levels up)
**Text overlay**: "AI Review • Instant Payouts • Reputation"

### [22-27s] Community Stats
**Visual**: Numbers counting up rapidly
**Text overlay**: "1000+ Bounties | $500K+ Paid | 500+ Hunters"
**Animation**: Numbers scroll up, emerald glow effect

### [27-30s] Call to Action
**Visual**: SolFoundry logo + URL
**Text overlay**: "Start Earning Today → solfoundry.xyz"
**Animation**: Subtle pulse glow, forge embers in background
