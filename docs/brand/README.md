# SolFoundry Brand Guide

Official brand guidelines for SolFoundry platform.

## Logo

### Primary Logo
- Full color logo for light backgrounds
- Minimum size: 120px width
- Clear space: 2x logo height on all sides

### Logo Variations
- **Dark version**: For light backgrounds
- **Light version**: For dark backgrounds
- **Icon only**: For favicon and small spaces

## Color Palette

### Primary Colors
- **SolFoundry Green**: #4EC9B0
- **Dark Background**: #1E1E1E
- **Light Text**: #D4D4D4

### Secondary Colors
- **Accent Blue**: #569CD6
- **Warning Yellow**: #DCDCAA
- **Success Green**: #6A9955
- **Error Red**: #F44747

## Typography

### Primary Font
- **Font Family**: Inter, -apple-system, sans-serif
- **Weights**: 400 (regular), 600 (semibold), 700 (bold)

### Usage
- **Headings**: Inter Bold, 24-48px
- **Body**: Inter Regular, 16px
- **Code**: JetBrains Mono, 14px

## Usage Guidelines

### Do's
- Use logo with adequate clear space
- Maintain aspect ratio
- Use approved color combinations

### Don'ts
- Don't stretch or distort logo
- Don't use unapproved colors
- Don't place logo on busy backgrounds

## Brand Voice

- **Professional** but approachable
- **Technical** but accessible
- **Innovative** and forward-thinking
