
# Searching and Filtering Bounties

This guide explains how to use the search bar and filters to find specific tasks and bounties in SolFoundry.

## Using the Search Bar

The search bar allows you to quickly find bounties by keyword:

1. Navigate to the bounties page
2. Type your search term in the search bar
3. Press Enter or click the search button

The system will display all bounties that match your search term, including:
- Bounty titles
- Bounty descriptions
- Tags associated with the bounty
- Contributor names

## Available Filters

You can refine your search results using the following filters:

### 1. Status Filter
- **Open**: Shows bounties that are currently available for claiming
- **Claimed**: Shows bounties that have been claimed by contributors
- **Completed**: Shows bounties that have been fully completed
- **Rejected**: Shows bounties that have been rejected

### 2. Category Filter
Filter by bounty category to find bounties in specific areas:
- Development
- Documentation
- Testing
- Research
- Design
- Community

### 3. Priority Filter
Filter by bounty priority level:
- Low: Bounties with minimal requirements
- Medium: Standard bounties with moderate complexity
- High: Important bounties with significant impact

### 4. Date Range Filter
Filter by creation date to find recent or older bounties:
- Last 7 days
- Last 30 days
- Last 90 days
- All time

## Combining Filters

You can combine multiple filters to narrow down your search results. For example:
- Find all "High" priority bounties in the "Development" category that were created in the last 30 days
- Find all "Open" bounties with the "Testing" tag

## Advanced Search Tips

1. Use quotation marks for exact phrase matching: `"smart contract audit"`
2. Use the `-` symbol to exclude terms: `solana -token`
3. Combine multiple terms with AND/OR logic: `development OR documentation`
4. Use wildcards for partial matches: `bount*`

## Finding Specific Bounties

To find a specific bounty:
1. Use the search bar with the bounty title or ID
2. If you know the bounty ID, you can directly navigate to it using the URL format: `/bounties/{id}`

## Sorting Results

You can sort your search results by:
- Relevance (default)
- Creation date (newest first)
- Creation date (oldest first)
- Bounty amount (highest first)
- Bounty amount (lowest first)
