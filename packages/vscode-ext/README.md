# SolFoundry Bounties - VS Code Extension

Browse and manage SolFoundry bounties directly within VS Code.

## Features

- **Bounty Explorer**: View all available bounties in the sidebar
- **Tier Organization**: Bounties organized by reward tier (T1, T2, T3)
- **Detailed View**: Click any bounty to see full details
- **Quick Actions**: Refresh, search, and filter bounties
- **Webview Panel**: Rich HTML interface for browsing bounties

## Installation

1. Install from VS Code marketplace
2. Or install from VSIX

## Usage

1. Open Command Palette (Cmd+Shift+P)
2. Type "SolFoundry: Open Bounties"
3. Browse available bounties

## Commands

- `SolFoundry: Open Bounties` - Open bounty browser panel
- `SolFoundry: Refresh Bounties` - Refresh bounty list

## Development

```bash
npm install
npm run compile
```

## Requirements

- VS Code 1.74.0 or higher
