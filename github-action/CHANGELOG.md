# Changelog

## [1.0.0] - 2026-04-22

### Added
- Initial release of SolFoundry Bounty Creator GitHub Action
- Label-triggered bounty auto-posting
- Tier detection from labels (tier-1, tier-2, tier-3)
- Custom reward amounts via label or issue body
- Skill extraction from issue labels
- Dry-run mode for testing
- Auto-comment on issue with bounty link
- Configurable deadline, reward token, and default tier
- Full TypeScript source with type definitions
- Comprehensive test suite
